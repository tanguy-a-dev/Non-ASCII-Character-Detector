# Privacy Policy — Non-ASCII Character Detector
*Last updated: June 2026*

This extension does not collect, store, transmit, or share any data of any kind.

All processing happens locally in your browser. The extension reads the text content of the current page solely to detect non-ASCII characters and display the result in an overlay. Nothing is sent to any server. No analytics, no telemetry, no logging.
The only data written to storage is a single boolean value recording whether the extension is enabled or disabled. This value never leaves your device.
